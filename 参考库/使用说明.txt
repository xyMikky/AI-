# 参考库 · 使用说明（V2.0）

## 参考库是什么

参考库是你的**个人设计参考基因档案系统**，通过持续积累参考图片和感受记录，形成一套 AI 可直接读取的个人参考偏好数据库。当你使用 AI 设计师助手处理创作任务时，AI 会按需加载相关领域的档案，让每次建议都符合你独特的审美，而不是给出千篇一律的"大众化"方案。

## 如何学习一张图片（参考学习模式）

### 基础用法

在对话中上传图片，并说：

```
@参考库主控助手.txt
[上传图片]
学习这张图
```

AI 会自动：
1. **判断领域**：这张图属于哪个子文件夹（A-J 或 Z）
2. **按领域框架分析**：不同领域使用专属的分析维度
3. **输出结构化记录**：生成可直接复制粘贴的格式化文本
4. **告知归档路径**：明确告诉你把记录粘贴到哪个文件

### 批量学习

```
@参考库主控助手.txt
[上传多张图片]
批量学习这几张图，逐一分析
```

### 记录不喜欢的设计

```
@参考库主控助手.txt
[上传图片]
把这个加入禁忌风格，这是我不喜欢的类型
```

### 用文字描述代替图片

```
@参考库主控助手.txt
我看到一款产品包装特别好看：深绿色背景、烫金字体、极简构图，
有很强的高级感。请帮我记录这个偏好到空间与产品领域。
```

## 如何归档记录

AI 分析完成后，会输出：
- 完整的结构化记录文本
- 明确的归档路径提示

你只需将记录文本**复制粘贴**到对应档案文件的「喜爱记录」区域即可。

同时建议：
- 更新 `参考索引.txt` 中对应领域的记录数量和偏好标签（保持索引准确）

## 如何在设计任务中调用参考库

参考库有两种调用方式：

### 方式1：通过主控中心自动调度（推荐）

主控中心在处理视觉创作任务时会自动判断是否调用参考库，并通过参考库主控助手精准决策加载哪几个子档案：

```
@主控中心.txt @参考库主控助手.txt @B_配色系统/配色参考档案.txt @Z_禁忌案例/禁忌风格档案.txt
帮我为一个咖啡品牌设计配色方案
```

### 方式2：手动指定档案（精准控制）

你可以直接指定需要加载的领域档案：

```
@主控中心.txt @M1_品牌与Logo设计.txt @A_品牌与Logo/品牌参考档案.txt @B_配色系统/配色参考档案.txt @Z_禁忌案例/禁忌风格档案.txt
设计一个餐饮品牌的Logo
```

### 方式3：轻量模式（只加索引）

当想要快速灵感参考，而不需要精确约束时：

```
@参考库主控助手.txt @参考索引.txt
基于我的整体审美风格，为这个项目提供风格方向建议
```

## 精准调用速查表

| 设计任务 | 建议加载的档案 |
|----------|---------------|
| Logo / 品牌设计 | `A_品牌` + `B_配色` + `Z_禁忌` |
| 完整品牌体系 | `A_品牌` + `B_配色` + `C_排版` + `Z_禁忌` |
| UI / APP设计 | `D_UI界面` + `B_配色` + `Z_禁忌` |
| 海报 / 平面物料 | `E_平面` + `B_配色` + `Z_禁忌` |
| 含摄影的海报 | `E_平面` + `B_配色` + `F_摄影` + `Z_禁忌` |
| 插画 / 图标 | `G_插画` + `B_配色` + `Z_禁忌` |
| 3D视觉 | `H_3D` + `B_配色` + `Z_禁忌` |
| 品牌视频 / 动效 | `I_动效` + `B_配色` + `Z_禁忌` |
| 包装设计 | `J_空间产品` + `A_品牌` + `B_配色` + `Z_禁忌` |
| 风格灵感 | `参考索引.txt`（全局浏览） |

## 生成参考摘要

当某个领域积累 5 条以上记录时，可以让 AI 自动生成该领域的偏好摘要，填入档案顶部的「领域偏好摘要」：

```
@参考库主控助手.txt @D_UI界面/UI参考档案.txt
请基于我的记录，生成UI领域的参考偏好摘要
```

---

## 扩展新领域

如果你有新的参考领域（如「国风文化」「潮流街头」「工业机械」等），可以自行新建文件夹：

1. 在 `参考库/` 下新建文件夹，命名格式：`K_[领域名]/`
2. 在文件夹内新建 `[领域名]参考档案.txt`，参考现有档案格式编写
3. 在 `参考索引.txt` 中新增对应条目
4. 告知 AI：「参考库新增了 K_[领域名] 领域，对应档案路径是 K_[领域名]/[领域名]参考档案.txt」

## 维护建议

| 频率 | 操作 |
|------|------|
| 随时看到好设计 | 截图保存，备注感受 |
| 每周 1-2 次 | 上传积累的截图，触发 AI 分析，将记录粘贴归档 |
| 每月 1 次 | 更新索引，回顾并清理不再符合当前审美的记录 |
| 每季度 1 次 | 生成各领域摘要，更新全局参考基调描述 |
