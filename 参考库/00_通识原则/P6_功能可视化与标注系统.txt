## P6-001 · 功能可视化是工具集，而非固定格式

**原则**：发光效果、虚线标注、解剖图、开盖展示、箭头指向、同心圆扩散、数字 badge——选哪种取决于品牌语气和产品复杂度。功能可视化不是一种风格，而是多种视觉工具的灵活组合。

**交叉验证**：E5b1#069（功能可视化工具集概念） + E5b1#081（同一页只用一种标注技法不混用） + E6b3a1（几何图形可视化无形功能） + D1e（节点编辑器条件逻辑可视化） + N 通用（详情页"功能可视化叙事"标签：剖面/爆炸图/虚线路径/同心圆按产品品类组合调用） + N2c2a（净水详情页水流路径动态 CGI） + N2b2（美容仪 RF 射频能量波纹 + 温感粒子可视化）
**覆盖域**：E × D × N = 3 域 ✅

## P6-002 · 箭头/方向标注的情感差异

**原则**：红色直线箭头 = 施力方向/物理原理（最低成本功能标注工具）；白色曲线弧形箭头 = 三维曲面身体上的托举包裹感（更柔和）；橙色箭头 = 主动重塑能量感。细线指向（不用箭头）= 高端轻盈感，避免廉价感。

**交叉验证**：E5b1（红色箭头翻转对比不可见功能可视化） + E5b1（白色曲线弧形箭头三维曲面施力感） + E5b1（细线指向避免廉价感） + E6b3a1（带方向箭头传达运动动态功能）
**覆盖域**：E = 1 域（但标注系统通用性强，保留）

## P6-003 · 圆形放大 + 引线 = 细节工艺展示

**原则**：对于细节工艺卖点在全景图中不可见的产品（肩带/接缝/垫子/面料纹理），圆形气泡放大 + 引线是最直接的解决方案。三个标注点应精准对应「目标买家下单前最可能的疑虑」。

**交叉验证**：E5b1（圆形面料特写 + 功能描述左侧标注系统） + E5b1（圆形气泡放大 + 引线细节工艺） + E5b2（数字 badge 系统说明功能区域分布） + E5a（2×2 特写格展示 4 个品质点）
**覆盖域**：E = 1 域（但标注通用，保留）

## P6-004 · Before/After 对比的执行规范

**原则**：Before/After 是最高转化格式之一，但只有在「变量只有产品本身」时才有说服力——同一模特同一天、同一光线同一站姿。After 叠加关键功能标注 ≤2 个。差异化底色（冷中性 vs 暖正向）强化情绪反差。

**交叉验证**：E5b2（Before/After 差异化底色强化反差） + E5b2（侧面局部无头部是功能最优视角） + E4（After→Before 逆向叙事先展示解决状态） + E6b3a1（清洁前后对比必备视觉元素）
**覆盖域**：E = 1 域（电商核心工具，保留）

## P6-005 · 无形功能的几何可视化

**原则**：覆盖范围 → 同心圆/波纹扩散；声场 → 扩散波纹；气流 → 粒子流动路径；净化 → 蓝色全息地图；智能规划 → 路径轨迹可视化。让消费者用图形理解抽象功能，比文字描述效率高 5-10 倍。

**交叉验证**：E6b3a1（同心圆波纹可视化功能覆盖范围） + E6b2a（蒸汽光效热感可视化） + E6b3a1（蓝色全息地图 AI 智能清扫可视化） + E6b2a（音箱声场同心圆扩散可视化） + D1d（数据管线流程条件可视化）
**覆盖域**：E × D = 2 域 ✅

## P6-006 · 功能说明面积 ≤ 30%

**原则**：功能说明区域应控制在广告总面积的 20-30% 以内，防止说明书感破坏情感吸引力。功能图标（小插画 + 短标题）是让不可见卖点可见的高效工具。多个标注应覆盖不同层级的用户顾虑。

**交叉验证**：E5b1#070（功能说明区域 20-30%） + E2b2（信息架构模拟用户购买决策顺序） + E2b1（功能标注图对体验优先产品极有价值） + P1-001（情感 70-80% + 功能 20-30%）
**覆盖域**：E = 1 域（与 P1-001 交叉验证）

## P6-007 · 图标矩阵的扫读设计

**原则**：5-6 格图标矩阵，每格「图标 + 关键词 + 核心数字」支持 1 秒扫读。图标风格全统一（全线描或全填充不混用）。使用「感觉类」图标（透气 = 波浪线/弹力 = 箭头延展）而非「功能类」图标。

**交叉验证**：E6b3a1（5-6 格图标矩阵 1 秒扫读） + E5b1#082（图标风格统一不混用） + E5a（弹性伸展功能用简单线条符号降低跨语言门槛） + D1a（图标语义一致性）
**覆盖域**：E × D = 2 域 ✅

## P6-008 · 材质成分百分比信息卡

**原则**：材质成分百分比 + 图标圈信息卡是功能性服装/护肤品/运动装备 A+ 内容的标配页。成分百分比字号 ≥48pt（移动端缩略图可读）。同页图标风格统一。

**交叉验证**：E5b1#082（材质成分百分比信息卡标配） + E5a（面料功能标题覆盖功能性 + 舒适性） + E5b2（尺寸指南浅底色 + 黑色线描简约风格）
**覆盖域**：E = 1 域（电商核心工具，保留）

## P6-009 · 幽灵图/平铺图 = 结构信任

**原则**：幽灵平铺图（无人穿着的产品平铺）是展示结构设计的必备视图。模特图展示穿着效果（情感），幽灵图展示结构设计（理性）。建议放在详情页产品参数区段，目标是已被种草但需技术确认的决策阶段买家。

**交叉验证**：E5b2（幽灵平铺图必备补充视图） + E5b2（正背面双视图 + 虚线分区框 + 数字 badge） + F1a（白底三视角系统正/3/4/背） + E2b2（多色系幽灵图并排跨色系差异展示）
**覆盖域**：E × F = 2 域 ✅

## P6-010 · 工作流/流程的节点化可视化

**原则**：多步骤流程（审批/数据管线/购买决策/使用步骤）用节点 + 连线 + 条件分支可视化。节点内可嵌数据预览；连线上贴条件标签；端口用左入右出语法。适用于 UI 设计、产品使用说明、营销漏斗可视化。

**交叉验证**：D1a（营销自动化流程三栏节点布局） + D1d（数据管线 ML 节点编辑器） + D1e（iPaaS 可视化图谱节点编辑器） + D1b3（协同可视化流程分析）
**覆盖域**：D = 1 域（但可视化原则通用，保留）

## P6-011 · 数据对比的可视化工具选择

**原则**：条形图/雷达图 > 纯文字数字列表（技术参数对比）。百分比环/进度条 > 文字描述（功能覆盖度）。「花 X 元得 Y 价值」> 模糊展示赠品图片（促销价值量化）。凡能量化的，用视觉量化比文字描述说服力高 3 倍。

**交叉验证**：E6b2b2（技术参数对比条形图/雷达图可视化） + E6b4a（量化花 X 元得 Y 价值） + E6b3a2（功能指标数据技术接口可视化） + D1d（数据分析仪表盘可视化）
**覆盖域**：E × D = 2 域 ✅

---

## P6-012 · A+ 内容页的系统化设计

**原则**：A+ Feature Page 适用于任何有 3+ 独立核心功能的产品。标题 serif + uppercase + 大到缩略图可读；每页文字 ≤30 词；标注技法选择：小硬件 → 特写嵌入，内部结构 → 虚线路径，形变方向 → 曲线箭头，同一页只用一种。

**交叉验证**：E5b1#081（A+ Feature Page 系统） + E5b1#082（材质成分百分比信息卡标配页） + E2b2（全套 A+ 内容纵览图应用场景）
**覆盖域**：E = 1 域（Amazon 电商核心工具，保留）
