## P1-001 · 情感主体与功能信息的面积权衡（7:3 法则）

**原则**：情感吸引面积（主图/模特/场景）应占 70-80%，功能验证面积（标注/说明/数据）应占 20-30%。先建立视觉吸引力，再提供理性验证。功能说明 > 40% → 变成说明书。

**交叉验证**：E5b1#069（功能标注图面积分配） + E5b1#070（虚线标注+促销并存） + E2b1（模特+产品卡比例） + F2b（产品摄影主次分配） + J2（家电产品图信息层级）
**覆盖域**：E × F × J = 3 域 ✅

## P1-002 · 留白越大，主体越有重量

**原则**：大面积留白 + 极小主体，是制造「天人合一」「孤独」「仪式感」的核心构图逻辑。天空/背景占 70% 时，地面 30% 的人物/产品却成为整幅画的视觉重心。越空旷，画面越有重量。

**交叉验证**：E1a1b（水墨留白 + 极小人物） + E1a3c2a（航拍风景 + 白色大字） + F1c2b2（约塞米蒂峡谷消失点） + F1c2b1a（盐湖极简前景） + H2a（科技极简景观） + G2（水墨留白）
**覆盖域**：E × F × H × G = 4 域 ✅

## P1-003 · 尺度颠覆：巨物 vs 渺小人物

**原则**：极小人物 + 极大物体（文物/山脉/产品/建筑/生物）是传达「个体与宏大力量关系」最有效的构图语言。尺度差距越大，叙事张力越强。适用范围：品牌海报、产品宏观叙事、文化项目、旅行内容。

**交叉验证**：E1a1a#001（文物纹理 + 微小人影） + E1a3a3（巨型元素 ≥70% 画幅） + E1a3c2b（文化符号俯视压迫） + F1c2a1（渺小人物 + 超大自然） + F1c2b2（峡谷消失点） + H2b（幻想世界人物渺小感）

## P1-004 · 框架构图：拱门/圆框/窗框作为叙事入口

**原则**：「通过某个物件的框/洞看主体」是制造「进入某个世界」「仪式感」「第三人旁观」的经典构图法。框架可以是拱门、圆圈、窗框、镜面、眼镜框、山洞入口。框架本身承载叙事功能。

## P1-005 · 对角线分割制造动态张力

**原则**：对角线/斜向分割比垂直/水平分割更有视觉动感和张力。适用于：促销图的信息分区、产品飞出构图、时尚品蕾丝/纹理展示、色彩版面对比叙事。斜向排布比垂直排布在视觉上更有运动感。

**交叉验证**：E5b3（对角线分割版式） + E2a2（对角色块布局） + E6b1b1（对角线飞出构图） + F2b（斜向对角线构图） + F2d（对角线斜置构图水面场景）
**覆盖域**：E × F = 2 域 ✅

## P1-008 · 背影构图的代入感优势

**原则**：背影（去除面部细节）让任何观看者都能代入，比正面肖像更能激发共情和想象空间。「背影 + 远方目标（光源/建筑/地平线）」= 旅程/坚持/宿命叙事。背影 + 手指指向 = 双效视线引导。

**交叉验证**：E1a1b（背影人物 + 水墨流动） + E1a3b1（室内逆光背影青春感） + F1c2a1（背影 + 手指巨云） + F3b（背面视角面朝光源） + F1c2b2（宠物背影面向开阔场景） + H2b（背影人物废墟宏大对称）

## P1-009 · 三段式纵向/横向结构

**原则**：「天 + 地 + 人」或「情感 + 触发 + 理性」三层/三区结构，是建立叙事层级感、避免混乱的万能布局。电商版式中体现为「形象区 + 促销区 + 产品区」，海报中体现为「天空 + 场景 + 人物」。

**交叉验证**：E1a1b（天地人三层群像） + E2b2（大生活图 + 产品格 + 促销条三层） + E6b2b2（情感插图 + 实物信任 + 数据说服三层） + F1c2a2（三层色彩系统） + N 通用（详情页三段骨架：Hero 情感区 + 功能验证区 + 转化区）+ D1a（UI 三栏布局：菜单 + 画布 + 属性）

## P1-010 · 仰拍 = 英雄感 / 俯拍 = 全知视角

**原则**：相机角度即叙事立场。平视略仰 = 产品「登顶」英雄感；完全仰拍 = 力量/解放/自由；俯视 = 全知/控制/策略；平视 = 平等对话。选择角度前先问「观者应该站在什么立场看这个主体？」

**交叉验证**：E1a3a3（视角即立场） + F2a1（平视略仰英雄感） + F1b1（低角度仰拍廓形展示） + E6b1b2（仰角拍摄戏剧光影） + J3（悬浮戏剧感构图）

## P1-011 · 左右分区版式：感性 vs 理性

## P1-013 · 「1 主 + N 辅」的面积层级

**原则**：当需要展示多个元素时，一个元素占据绝对主导面积（≥60%），其余元素以显著更小的面积辅助。主辅面积差距越大，信息层级越清晰。适用于：1 hero + N 副产品、主图 + 缩略产品卡、大场景 + 小标注。

**交叉验证**：E2a1（1 hero + N 副产品） + E5b3（1 主 + N 辅版式） + E2b1（生活方式主图 + 产品缩略叠加） + F1a（单一主色在中性背景中权重远超面积）
**覆盖域**：E × F = 2 域 ✅

## P1-019 · 全出血英雄图 vs 分割面板：视觉重量的根本差异

**原则**：Hero 图 / Banner / A+ 首屏中，**全出血摄影**（照片铺满画幅 + 文字叠印在构图自然留白区）的视觉重量远强于**分割面板**（左照片 / 右白色文字区）。分割面板将版面割裂为「摄影作品」与「印刷品」两块，消减了摄影本身的情感张力；全出血让摄影语言统治全局，文字成为画面的一部分而非对立面。

执行要点：
- 模特站位应为构图留白自然腾出空间（模特右置 → 左侧自然留白；或全幅满构图配超小型文字叠印）
- 模特朝向与文字区形成视线对话，引导视线流
- 文字叠印区的背景必须足够单纯（浅色墙 / 纯色区域），保障文字可读性
- 分割面板仅适合信息密度高的非英雄功能模块（标注图、尺码表、图标矩阵）

**交叉验证**：NEBILITY AC3383A A+ Hero 迭代实战（V1 分割面板 → V4 全出血进化路径） + E2a2（生活方式主图+浮层促销保留品牌感） + P1-011（左右分区感性vs理性）
**覆盖域**：E（实战高确信度推导）= 1 域 ⚠️（待第二域验证升级）

## P1-020 · 功能近景裁切：让产品功能区占领视觉中心

**原则**：当产品的核心功能高度集中于特定身体部位（塑身衣=腰腹/臀；运动内衣=肩背；压缩裤=腿部），**半身特写裁切**（将该功能区放大至画面 ≥60%）比全身展示具有更强的产品说服力。全身构图将产品面积稀释，功能细节难以被感知；近景裁切将「产品如何工作」直接映射为「视觉如何工作」——消费者用眼睛就能感受到收腰、提臀、拉伸等效果。

品类近景裁切参考基准：

| 产品品类 | 裁切范围 | 核心展示区 |
|---|---|---|
| 高腰塑形短裤 | 乳下 → 大腿中部 | 高腰带压缩效果 + 臀部轮廓 |
| 塑身连体衣 | 肩部 → 大腿上部 | 完整产品轮廓 + 背部/腹部线条 |
| 运动内衣 | 颈部 → 腰线 | 肩带支撑结构 + 背部设计 |
| 压缩打底裤/Legging | 腰带 → 小腿 | 腿部修形线条 + 面料纹理 |
| 胸衣/Bra | 颈部 → 腰线（或纯背面） | 支撑杯型 + 肩带设计 |

**交叉验证**：NEBILITY AC3383A A+ Hero V4 半身特写迭代 + P1-014（产品展示的最优视角选择）+ E5b2（侧面局部最优功能视角）
**覆盖域**：E（实战推导，与 P1-014 互补——P1-014 讲镜头角度，P1-020 讲裁切范围）= 1 域 ⚠️（待第二域验证升级）
