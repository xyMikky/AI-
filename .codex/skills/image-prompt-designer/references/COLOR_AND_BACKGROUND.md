# 色彩纪律与背景策略

## 核心原则

画面中任一像素的颜色，都应能回答两个问题：**① 它属于什么角色？② 它的颜色来自哪张权威图？**

当一张画面同时含**色卡图**与**版式/风格参考图**时：**色卡是唯一颜色权威**；版式/风格图只提供结构、布局、装饰几何，**不提供**产品色、肤色或整体照片色调。

**禁止**以"让照片与参考图色调协调"为由，对产品照片、模特皮肤、产品特写做降饱和/偏色/色调拉齐。产品色永远是产品色，背景色永远是背景色，装饰色永远是装饰色——三者各司其职，互不污染。产品色与装饰色的**对比**本身就是高级感来源，不是"不协调"。

---

## 参考图的色彩权威分层

| 参考图类型 | 提供的色彩权威 | 不提供 |
|---|---|---|
| 产品图 | 产品固有色、模特肤色发色、面料/缝线颜色 | 背景色、装饰色、整体色调 |
| 版式/风格图 | 背景色、装饰几何色块色、空间构图 | ❌ 产品色 ❌ 肤色 ❌ 产品特写的照片色调 |
| 品牌资产图 | 品牌字体色、品牌主色、文字排版 | 背景色、产品色、装饰色 |
| 色卡图（权威） | 所有 HEX + 角色 + 占比，最终裁决 | 不提供图形结构 |

裁决优先级（信号冲突时）：`色卡图 > 产品图固有色 > 品牌资产图字体/品牌色 > 版式图装饰色（仅限装饰元素）`。

---

## 色彩角色隔离（Color Role Isolation）

每个颜色必须标明所属元素范围，角色与像素是**硬约束**：

| 色卡角色 | 只可用于 | 不可用于 |
|---|---|---|
| `BACKGROUND` 背景色 | 整块空白画布、大面积底色 | 产品、肤色、装饰 |
| `PRODUCT` 产品色 | 产品本体每个像素（主体+缩略图+特写） | 背景、装饰、文字 |
| `DECORATIVE ELEMENTS` 装饰色 | 几何块/圆/胶囊/分隔条/图标底 | 产品、肤色、照片特写 |
| `TEXT` 文字色 | 文字层 | 产品、装饰块 |
| `SKIN/NEUTRAL` 肤色/中性 | 模特皮肤、非核心中性单品 | 背景、装饰、核心产品 |

关键推论：产品特写缩略图中的**产品像素**必须用 PRODUCT 色，不能用 DECOR 色"统一色调"；不同位置出现的同款产品，视觉上必须是同一种颜色。

---

## Prompt 硬约束句式

> `image N`、HEX、大写标记词（`BACKGROUND ONLY` 等）是技术锚点，保留原样。

### 句式 A：产品色独立性（画面含产品照片时必写）
```text
产品色保护区（PRODUCT COLOR — PROTECTED ZONE）
画面中所有产品像素——无论在主体、产品细节方格还是任何特写缩略图中——精确呈现 image 1 中的原始产品色。
产品色独立于装饰色调，保持完全饱和，色相与 image 1 精确一致。
```

### 句式 B：色彩角色隔离（有色卡图时必写）
```text
色彩角色隔离（COLOR ROLE ISOLATION，源自 image N 色卡）
- #e8e4dd → BACKGROUND ONLY，仅用于背景底色。
- #8a8577 → DECORATIVE ELEMENTS ONLY，仅用于几何/分隔条/方格。
- #4b2e27 → PRODUCT ONLY，仅用于产品像素。
- #2a2a28 → TEXT ONLY，仅用于标题/标签/说明文字。
```

### 句式 C：特写防色调牵引（有产品特写/小缩略图时必写）
```text
特写产品色保真（DETAIL THUMBNAIL COLOR FIDELITY）
右侧小方格产品特写在干净中性棚拍光下呈现真实产品色，色相与 image 1 一致、饱和度完全保留，独立于装饰色调体系。
```

## 禁句清单（提交前必查）

出现下列任一短语（无论中英文）即违反纪律，**必须改写**：

| 禁句 | 问题 | 正向替代 |
|---|---|---|
| `色调向…统一以协调画面` / `color graded toward … to harmonize` | 产品色被迫向装饰色靠拢褪色 | `产品色精确呈现 image N 原色，棚拍光下完全饱和` |
| `降饱和以匹配…` / `desaturated to match the palette` | 强制产品降饱和 | `产品色保持完全饱和，色相与 image N 一致` |
| `与背景色调融合` / `hue shifted toward the scene` | 产品色受背景牵引 | `产品色独立于背景色，保持自身色调与饱和度` |
| `所有元素色彩统一` / `cohesive color story`（未声明产品例外） | AI 默认会降饱和产品 | `装饰元素共享低饱和色调；产品色独立保持饱和原色，二者形成张力` |
| `调成编辑感色调`（未明确产品例外） | 编辑感调色=产品褪色 | `编辑感氛围由装饰色调与光线构建；产品色保持原色作为饱和焦点` |

## 背景色策略（可配置）

> 这是**项目可配置**的策略，不锁死。下方是常见电商默认；具体项目可按品牌/用户偏好覆盖。

### 默认策略（电商常用）
- 背景默认纯白 `#ffffff`；需要层次时用极浅灰白 `#fafafa`–`#f5f5f5`。
- 整体明亮、高调（high-key），主体清晰。
- 大面积（占比 ≥30%）避免中低明度灰/灰黄/灰绿/暖米/杏仁米等"显旧"色，除非有品牌或用户明确指令。

### 不盲目继承参考图背景
参考图背景是非白色（暖米/灰绿等）时，**不要默认继承**。先确认：
```
参考图背景是 [色系]（#hex）。你希望：
  A. 保留参考图的 [色系] 背景
  B. 改用纯白 #ffffff
  C. 其他指定色
```

### 上位覆盖优先级
`品牌规范明确色值 > 用户本轮明确指令 > 本策略默认`。当背景偏离默认时，在交付说明里注明"本次背景为 [色值]（来源：[品牌/用户指令]），偏离默认白底"。

### 默认背景 Prompt 句式（无上位指令时）
```text
背景——干净的纯白背景，精确使用 #ffffff。明亮、清爽、高调的摄影背景，均匀中性的棚拍灯光。
整体色彩基调——明亮、高调、清爽现代。产品色是白色画布上的饱和视觉焦点。
任何低饱和/大地色调仅允许作小面积点缀（< 20%），不得作主色或背景色。
```
