# K1 · 中国大陆市场设计规范

> 场景标签：K1_中国大陆
> 适用任务：面向中国大陆用户的一切设计产出

---

## 视觉语言特征

### 色彩偏好
- **红色**具有强烈正面联想（喜庆、吉祥、促销成功感），大促期间高频使用
- **金色**传达高端感和品质，常与红色搭配
- **白色**在大多数场景是背景色首选，整洁感
- **高饱和度**在大众消费品电商中被证明有效（视觉抢眼），但高端品牌反之
- 低饱和/莫兰迪色系适用于：美妆、家居、轻奢、女性向品牌

### 视觉密度
- 大众电商平台（淘宝、拼多多）：**高密度**，信息量大，用户习惯扫描式阅读
- 高端/品牌化（天猫旗舰店、品牌官网）：**中等密度**，留白增加但不极简
- 极简留白：适合定位高端、面向一线城市年轻白领的品牌

### 字体使用
- 黑体、方正系列：主流，识别性强
- 宋体：传统感、文化感
- 手写体：温暖、有温度感（餐饮、生活方式品牌）
- 英文字体：现代感、国际化定位标配

### 促销信息呈现
- 「低价直击」视觉语言：大字报式、红底黄字、数字醒目
- 「品质溢价」视觉语言：克制呈现价格，强调产品故事和材质
- 节日营销：大促视觉必须呼应节日元素（春节红包、双11符号等）

---

## 文化敏感点

| 类型 | 说明 |
|------|------|
| 正向借力 | 龙、凤、福字、牡丹、国潮元素——恰当使用增加文化认同 |
| 需谨慎 | 绿色帽子（不忠联想）、伞（散/分离联想）、数字4（死谐音） |
| 绝对禁忌 | 台湾/西藏/香港政治相关表述、宗教污化 |

---

## 不同平台差异（中国大陆内部）

| 平台 | 用户心理 | 设计倾向 |
|------|---------|---------|
| 淘宝 | 价格敏感，货比三家 | 促销感强，信息密集，主图要醒目 |
| 天猫旗舰 | 品质意识较强 | 品牌感，视觉质感高于淘宝 |
| 拼多多 | 极度价格敏感 | 红色/橙色系，大字促销，简单直接 |
| 小红书 | 追求生活方式和种草 | 真实感、场景化、高颜值、文字简洁 |
| 抖音/快手 | 冲动消费，视觉刺激 | 动态感强，快节奏，强钩子视觉 |
| 微信/私域 | 信任关系 | 温度感，社群感，减少硬广感 |

---

## 有效设计模式（电商）

1. **主图**：白底或场景化二选一，产品占图面积 >60%，核心卖点文字叠加
2. **详情页顺序**：场景图 → 痛点提问 → 产品解决方案 → 细节展示 → 信任背书 → 促销行动
3. **信任背书视觉元素**：质检报告、资质证书、销量数字、用户评价截图
4. **促销价格展示**：划线原价 + 当前价，视觉对比强烈
