# K4 · 详情页设计规范（Product Detail Page Design Norms）

> 场景标签：K4_详情页设计
> 适用：各电商平台的产品详情页/描述页/A+ Content 的完整长页面设计
> 与「详情页主图规范」的区别：主图规范仅覆盖第一张主图，本规范覆盖从首屏到底部的全部模块

---

## 详情页的核心目标

**将「浏览者」转化为「购买者」。**

详情页是用户做出购买决策的最后一个环节。它必须在有限的滚动距离内完成：
1. **吸引**（3秒内传达核心价值）
2. **说服**（用功能/效果/场景回答"为什么买"）
3. **信任**（用证据回答"为什么信你"）
4. **行动**（降低行动门槛，给出明确指引）

---

## 中国 vs 欧美：两套完全不同的设计逻辑

### 欧美详情页设计逻辑

| 维度 | 特征 |
|------|------|
| 设计哲学 | Less is more — 每屏一个信息点 |
| 视觉密度 | 低，大量留白 |
| 文案风格 | Benefit-first，短句，可扫读 |
| 图文关系 | 图片主导，文字辅助 |
| 色彩 | 白底/浅色底，品牌色仅用于强调 |
| 信任逻辑 | 第三方背书 > 评论 > 数据 |
| 促销 | 克制，折扣信息不侵入产品展示 |
| 滚动长度 | 短（5-10屏），信息高度浓缩 |
| 平台代表 | Amazon A+ / Shopify PDP / DTC |

### 中国详情页设计逻辑

| 维度 | 特征 |
|------|------|
| 设计哲学 | 信息即说服力 — 宁多勿缺 |
| 视觉密度 | 高，密集排布 |
| 文案风格 | 痛点驱动 + 数字量化 + 情感共鸣 |
| 图文关系 | 图文一体化（文字嵌入图片） |
| 色彩 | 品类差异大（大促红金/护肤白绿/科技暗蓝） |
| 信任逻辑 | 销量 > KOL/明星 > 用户晒单 > 认证 |
| 促销 | 激进，促销信息贯穿全页 |
| 滚动长度 | 极长（10-30屏），节奏感靠视觉疏密交替 |
| 平台代表 | 天猫/淘宝/京东/拼多多 |

### 欧美详情页排版高级感规范（Typography Premium Standards）

> **核心问题**：AI 生图时文字容易过大、占据过多画面空间，导致画面廉价感。欧美高端详情页的文字应"点到为止"，让视觉元素（产品/场景/摄影）做主角。

**文字空间占比控制**：

| 模块类型 | 文字区域占比上限 | 说明 |
|---------|---------------|------|
| HERO 首屏 | ≤ 30% | 主标题+副标题+CTA，文字区不超过画面 1/3 |
| USP-ICON 卖点矩阵 | ≤ 25% | 图标为主，文字为辅；标题短句+极简说明 |
| CALLOUT 功能标注 | ≤ 20% | 标注文字小而精炼，产品本体占画面核心 |
| LIFESTYLE 场景图 | ≤ 10% | 场景图以视觉为主，仅保留品牌名/极简标语 |
| TECH 技术说明 | ≤ 30% | 技术图解以视觉化为主，文字标注精简 |
| BA-COMPARE 对比 | ≤ 15% | Before/After 标签+简短说明，让对比图说话 |
| SOCIAL-PROOF 信任 | ≤ 25% | 评论精选以关键引语为主，不铺满文字 |

**字体层级体系（Typographic Hierarchy）**：

```
H1 主标题 — 画面中最大文字，但不超过画面宽度的 60%
           字重：Bold / Semibold
           风格：clean sans-serif 或 elegant serif，字间距适度加宽（letter-spacing: 0.05-0.1em）
           示例：Tummy Control Shapewear Bodysuit

H2 副标题 — H1 字号的 40-50%
           字重：Regular / Light
           风格：与 H1 同族但更轻，或使用衬线体/非衬线体对比
           示例：Designed for all women with the ultimate balance

标签文字 — H1 字号的 25-30%，全大写（ALL CAPS），加宽字间距（0.1-0.2em）
          字重：Medium / Regular
          风格：精致的小型大写标签，点缀式出现
          示例：BODYSUIT · GLASS · JEANS

说明文字 — H1 字号的 30-35%
          字重：Regular / Light
          风格：可读性优先，行高 1.4-1.6
```

**高级感字体风格关键词（用于 AI 生图 Prompt）**：

```
基础高级感：  elegant typography, refined font design, generous letter-spacing
克制精致型：  understated text, minimal typography, text as accent not focal point
编辑杂志型：  editorial typography, magazine-quality type hierarchy, sophisticated layout
奢侈品调性：  luxury brand typography, thin elegant letterforms, ample white space around text
运动功能型：  clean geometric sans-serif, technical precision typography, restrained text weight
```

**❌ 排版禁忌（欧美详情页）**：
- 主标题占据画面 50% 以上面积（信息过载，廉价感）
- 所有文字层级使用同一字号/字重（缺乏层次感）
- 文字挤满画面边缘，无呼吸空间（至少保留 10-15% 的文字区域边距）
- 使用过粗的 Ultra-Bold 字体充满画面（大促/下沉市场做法，不适合欧美高端线）
- 多种字体混用超过 2 种字族（1 种无衬线 + 1 种衬线为上限）

### 设计不可混用的硬性边界

| 做法 | 在欧美 | 在中国 |
|------|--------|--------|
| 密集叠字促销 | ❌ 被视为 spam | ✅ 常规做法 |
| 极简留白 | ✅ 高端感 | ❌ 信息不足，"不用心" |
| 红色大字促销 | ❌ 廉价感 | ✅ 大促标配 |
| 纯英文文案 | ✅ 本地化 | ❌ 疏离感 |
| 明星/KOL 大图 | ⚠️ 谨慎使用 | ✅ 核心信任手段 |
| 用户真实照片UGC | ✅ 强信任 | ✅ 强信任 |
| Before/After 对比 | ✅ 效果直观 | ✅ 效果直观 |

## 详情页设计节奏原则

### 视觉密度节奏（适用于中国长卷轴详情页）

```
[密] 首屏 HERO + PROMO — 信息密集，制造紧迫感
 ↓
[疏] USP-ICON — 图标+短文字，呼吸空间
 ↓
[密] CALLOUT + TECH — 功能细节密集展示
 ↓
[疏] LIFESTYLE — 全出血大图，视觉休息
 ↓
[密] BA-COMPARE + SOCIAL-PROOF — 证据集中展示
 ↓
[疏] BRAND-STORY — 情感节奏调节
 ↓
[密] SIZE-GUIDE + SPEC-TABLE + GUARANTEE — 决策辅助信息集中
```

### 信息类型交替原则

```
感性（场景/故事）→ 理性（参数/对比）→ 感性（用户评价）→ 理性（规格/售后）
```

人脑对交替呈现的信息类型保持更高注意力。全部理性或全部感性都会导致注意力衰减。
