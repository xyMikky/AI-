# [使用者名] · 个人索引

## 快速导航

| 文件 | 用途 | 何时加载 |
|---|---|---|
| `个人档案.txt` | 身份 + 工作上下文 + 偏好速查 | 每次会话（强制） |
| `基础信息/岗位与职责.txt` | 岗位职责、OKR、日常节奏 | 工作规划 / 岗位相关任务 |
| `基础信息/个人偏好.txt` | 沟通、设计、工具偏好 | 设计任务 / 个性化输出 |
| `基础信息/技能与工具.txt` | 技能栈、常用软件、熟练度 | 学习成长 / 任务分工 |
| `基础信息/协作关系.txt` | 上下游、对接人、交接习惯 | 跨部门协作 |
| `工作日志/工作摘要.txt` | 当前状态 + 待办 + 近期重点 | 每次会话（强制） |
| `工作日志/日志索引.txt` | 全部日志条目索引 | 查历史 / 复盘 |
| `工作日志/按年/` | 按日期归档的原始日志 | 深度回溯 |

## 素材入口

- `00_待学习/` — 个人简历、岗位说明、偏好清单等待解析素材
- `工作日志/00_待归档/` — 新工作日志粘贴区（单条或多条）

## 维护 Skill

- 个人档案：`profile-spec-learner`
- 工作日志：`work-log-manager`
