# 生成结果输出 · 统一输出根目录

本目录是系统**唯一的输出根目录**。所有 Skill / 工具生成的产物一律落在这里，禁止散落到项目根目录或其他位置。

## 子文件夹约定

| 子文件夹 | 存放内容 | 产出方 |
|---|---|---|
| `YYYYMMDD/` | 当日 AI 生图结果（按日期归档）| rh-image-pro-img2img / txt2img |
| `prompts/` | 每次生图的完整 Prompt 留档 | ai-image-prompt-builder |
| `color_palettes/` | 色卡 PNG | color-palette-generator |
| `brand_asset_sheets/` | 品牌资产合成图（Logo + 字体 + 色卡）| brand-asset-sheet |
| `font_renders/` | 字体渲染样张 | typography-designer |
| `visualizations/` | HTML 关系图谱 / 看板 / 流程图 | design-relation-visualizer 等 |
| `amazon图片提取/` | Amazon 套图 / A+ 抓取产物 | amazon-image-extractor |

> 新增产物类型时，先在本表登记子文件夹用途，再产出，保持目录可预期。

## 命名与清理

- 图片建议命名：`[标签]-[简述]-[YYYYMMDD].png`
- 本目录所有图片/HTML 已在 `.gitignore` 中忽略（产物可重生，不入版本库）；仅保留本说明与 `.gitkeep`
- 历史命名不一致的旧规则（曾用名「生成图片输出」）已于 V9.0 统一为本目录名「生成结果输出」
