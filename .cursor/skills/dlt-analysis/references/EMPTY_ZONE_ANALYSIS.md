# 空区（Empty Zone）方法论

> **第五分析角度**。空区 = 单期内一段**连续号码完全未开出**（参考图中的绿色横条）。直觉假设：空区有周期性，能预测下期空区即可排除约 1/3 号码、大幅提高准确率。**本文用 800 期回测对这一假设做了诚实验证，结论需要修正——见第 3 节。**

---

## 1. 两种空区定义（务必区分）

| 定义 | 含义 | 是否总存在 | 用途 |
|------|------|-----------|------|
| **A. 最宽空号段** | 单期内最长的连续未开号区间（如 25–34 连续 10 个号没开） | **几乎总存在**（5 球散落 35 号必留大缝） | 可视化（绿色横条）、描述「这期缺口在哪」 |
| **B. 严格三等分区空** | 01-12 / 13-24 / 25-35 中某一整区 0 球 | 仅 ~37% 的期出现 | 预测「排除整区」 |

参考图的绿色横条是**定义 A**（视觉直观）。而"预测排除一整区"用的是**定义 B**。两者必须分开评估。

## 3. 800 期回测的关键结论（诚实数据）

### 严格三等分区（定义 B）边际空区频率

| 分区 | 空区频率 | 说明 |
|------|---------|------|
| 01-12 | ~10.5% | |
| 13-24 | ~10.9% | |
| 25-35 | **~16.0%** | 该区仅 11 个号，结构上最常空 |
| **任一区有空** | ~37% | 多数期（63%）三区都有球 |
| **≥2 区同时空** | ~0.4% | 几乎不发生 |

### 各预测策略命中率（押 1 区为空，回测 800 期）

| 策略 | 命中率 | 评价 |
|------|--------|------|
| 正确随机基线（边际均值） | ~12.5% | 真实基准 |
| 最优固定押（永远押 25-35） | **~16.0%** | 结构性偏冷 |
| `max_due_ratio`（越久没空越该空）| ~10.6% | ❌ **低于基线，回归逻辑失效** |
| `coldest_k8`（近 8 期最冷区）| ~12.6% | ≈ 基线 |
| **押 2 个最冷区（覆盖更宽）** | **~25.8%** | 覆盖宽但不等于排除力强 |

### 结论

1. **"空区每 N 期规律出现且可精准预测"的强假设不成立**——严格整区为空是 ~10-16%/区 的弱事件，动态预测难显著超随机。
2. **`max_due_ratio`（最该出空区）方法被证伪**：命中率反低于基线，已从默认预测中移除。
3. **唯一稳定的弱信号是结构性偏冷**：25-35 区（11 个号）天然最常空（~16%）。
4. 本模块真正价值：
   - **描述性**：可视化每期空缺位置（绿框），辅助人工读图；
   - **结构提示**：指出最热活跃区（重点选号）与最冷区（适度降权）；
   - **不**提供高确定性的"排除一整区"能力。

> 这与「偏差值」程序的精神一致：用回测把直觉量化，让数据说话——结果不及预期时如实修正，而非强行声称有效。

## 4. 修正后的默认预测逻辑

`analyze_empty_zones()` 现采用：

- **首选空区** = 窗口内空区频率最高的区（结构性偏冷，常为 25-35）
- **两个最冷候选区** = 近 8 期命中球数最少的两区（更宽覆盖 ~26%）
- **最热活跃区** = 近 8 期命中最多的区（重点选号区）
- 对首选空区**弱降权**而非清零（单区严格全空仅 10-16%）

## 5. 与其他程序的关系

```
程序三 局部趋势 → 趋势落点 T
程序四 偏差值   → T ± N 扩展
程序五 视觉预测 → 人工读图
程序六 空区     → 调整各区权重：最热区多选、最冷区少选（弱信号，不强排除）
```

**定号时的正确用法**：

```
候选池（趋势落点 ± 偏差带 ∩ 冷热过滤）
  → 按空区结构微调：最热活跃区分配更多号，最冷区收敛号数
  → 不要因"预测空区"就把一整区完全排除（会误杀，因为整区全空仅 ~13%）
  → 最终固定收敛为 前区 7 + 后区 3（最热区权重 ×1.15、最冷区 ×0.85）
```

> `build_final_recommendation()`（`dlt_data.py`）已把上述空区弱权重内置进 7+3 定号流程：`local_trend_chart.py` 报告底部直接给出整合后的 7+3 候选池。

## 7. 常见误区

- ❌ **相信"预测空区可大幅提高准确率"而重仓排除整区** —— 回测显示整区全空仅 ~13%，强排除会高频误杀
- ❌ **用 `due`（越久没空越该空）逻辑** —— 已被证伪，低于随机
- ❌ **把绿框（最宽空号段）当成"下期必空区"** —— 绿框是历史描述，不是预测
- ✅ **把空区当弱权重信号**：偏向最热活跃区选号，对结构性偏冷区适度收敛
