# 偏差值（Deviation Value）方法论

> **第四分析角度**。纯趋势线延长落点的精确命中率约 **20%** 量级——不是因为趋势无效，而是因为**真实开奖多数在趋势方向正确的前提下，沿列轴产生 1–N 格的系统性偏移**。预测时必须把「趋势落点」与「偏差带」分开建模；偏差量需通过 `deviation_backtest.py` 持续回测、样本越大越准。

---

## 1. 为什么需要偏差值

| 现象 | 说明 |
|------|------|
| 趋势方向往往对 | 斜向链、汇聚枢纽能指示号码**大致运动区间** |
| 精确落点 rarely 命中 | 实际球常落在落点**左侧或右侧 1–5 列** |
| 纯趋势成功率 ~20% | 指「单球恰好等于链延长列号」的球级精确率（经验初值，以回测档案为准） |
| 加偏差带后覆盖率上升 | 落点 ±N 列的命中率显著高于纯落点 |

**结论**：趋势回答「往哪走」，偏差回答「走多远、偏几格」。

---

## 2. 偏差定义

对某一期实际开奖号 `A` 与当期全部趋势落点集合 `T`：

```
nearest = argmin_{t ∈ T} |A - t|
deviation = A - nearest        # 有符号列距（负=偏左，正=偏右）
abs_deviation = |deviation|
exact = (deviation == 0)
```

- **球级偏差**：5 个前区球 / 2 个后区球各自相对最近落点的列距
- **期级偏差**：该期是否至少 1 球精确命中落点（通常高于 20%，但仍偏低）
- **推荐偏差带 `±N`**：使 ≥80% 历史球落在 `[落点-N, 落点+N]` 的最小 N（由回测自动计算）

## 3. 与前三程序的关系

```
程序一 基本走势图（50 期）     → 全局结构
程序二 统计分析               → 冷热 / 欠账
程序三 局部趋势（15 期）       → 趋势落点 T
程序四 偏差值                 → 落点 ±N 扩展候选池
程序五 全局视觉预测（50 期）   → 人工读图 + 上述数字交叉
```

**定号公式（推荐）**：

```
最终候选 = 趋势落点 T
         ∪ (T ± recommended_band)
         ∩ 全局冷热 / 超期过滤
```

不可只押 `T` 本身。

| 字段 | 含义 |
|------|------|
| `front.exact_hit_rate` | 前区球级纯落点精确率 |

## 6. 常见误区

- ❌ **只买趋势线延长列** — 忽略 ~80% 会偏几格的事实
- ❌ **偏差带拍脑袋 ±3** — 应读档案 `recommended_band`，随回测更新
- ❌ **回测一次就固化** — 用户明确要求「大量验证来总结」，需迭代样本
- ❌ **把偏差带当成第二套趋势** — 偏差是围绕落点的**列轴微调**，不是新方向

---

## 7. 初值说明

用户经验：**纯趋势成功率约 20%**。回测脚本会用实际历史验证该量级；若样本偏差显著偏离 20%，以 `deviation_profile.json` 实测为准，并在档案 `notes` 中注明。

## 8. 7+3 定号附加信号（build_final_recommendation）

`build_final_recommendation()` 在趋势落点 + 偏差带之外，叠加两个均值回归信号：

| 信号 | 逻辑 | 参数 |
|---|---|---|
| **超期回补** | 遗漏 > 平均间隔（前 6 / 后 5）的号按超期幅度加分 | `(miss-avg) × max_score × 0.045` |
| **重号保护** | 上期开奖号叠加保护分，不被冷区回落剔除 | `+ max_score × 0.45` |

> 传入 `records=全量历史` 时用全历史遗漏值（更准）；否则退化为窗口内遗漏。

### ⚠️ 诚实回测结论（harness.py 盲测 300 期）

由 `scripts/harness.py` walk-forward 盲测（每期仅用 ≤ 基准期数据，预测落定后才揭晓），随机基线为蒙特卡洛同尺寸号池：

| 格式 | 指标 | 模型 | 随机基线 | 净差 |
|---|---|---|---|---|
| 7+3 | 前区命中/5 | 0.903 | 0.999 | **-0.096** |
| 7+3 | 后区命中/2 | 0.533 | 0.499 | +0.034 |
| 7+3 | 中奖率 | 14.0% | 17.3% | -3.3% |
| 15+5 | 前区命中/5 | 2.080 | 2.144 | -0.064 |
| 15+5 | 中奖率 | 63.7% | 65.2% | -1.5% |

**结论**：模型前区**全面未跑赢随机**，甚至略低（趋势收敛降低号池覆盖度）。后区仅 +0.034 的微弱正向，样本量下大概率是噪声。

**L2 看图盲测（补充 · strict 单期 100 期 `l2-100-single`）**：

| 格式 | 指标 | 看图法 | vs 随机 |
|---|---|---:|---:|
| 7+3 | 前区/5 | 0.860 | **-0.142** |
| 7+3 | 中奖率 | 15.0% | **-2.39%** |
| 15+5 | 前区/5 | 2.020 | **-0.118** |
| 15+5 | 中奖率 | 63.0% | **-2.02%** |

旧 run `l2-100`（10 期/批 subagent、答案未隔离）曾出现 +18% 中奖率虚高，**已作废**。L2 必须按 [VISUAL_HARNESS.md](VISUAL_HARNESS.md) 单期隔离才有可比性。

**这意味着**：超期回补 / 重号保护 / 偏差带等信号的价值是**让定号有逻辑可解释、结构更均衡**，而**非提高命中率**。单期出现 3/5、5/5 属方差，不可外推为可持续能力。

> 复核命令：`python scripts/harness.py --periods 300 --format both`。换 `--seed` / 加大 `--periods` 可验证后区微弱净差是否稳定（通常会回落到 ~0）。彩票每期独立随机，本工具仅供娱乐与数据分析练习。
