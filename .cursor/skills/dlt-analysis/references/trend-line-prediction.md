# 趋势线预测方法论（视觉分析）

> 本文件是「程序二：趋势线预测」的详细观察指南。Agent **必须先用程序一生成 50 期走势图 PNG，再用视觉能力读取该 PNG**，按本文流程逐步观察后给出预测。禁止不看图、仅凭 JSON 数字臆测。

---

## 0. 核心理论：均态分布与回归均值

每期号码随机产生，但**每个号码的出现概率相等**，因此在足够多的期数里，每个号码的出现频率会趋向一致——这就是「均态分布」。短期内必然出现偏离（某些号扎堆、某些号长期不出），而系统会通过后续开奖**向均值回归**。趋势线预测，就是用**视觉**在 50 期窗口里捕捉这种「偏离 → 回归」的运动方向。

> **推荐前置步骤**：在 Read PNG 之前，先运行 `stats_report.py -n 50` 获取超期冷号、热号、和值等数字摘要，作为 Step B/D 的交叉验证锚点——但最终定号仍以 PNG 视觉观察为主，stats 不可替代看图。

### 基准期望值（50 期窗口，务必记住）

| 维度 | 期望值 | 用途 |
|------|--------|------|
| 前区单号出现次数 | ≈ 7 次（50×5÷35） | 高于 7 偏热，低于 7 偏冷 |
| 前区单号平均出现间隔 | ≈ 7 期（平均遗漏 ≈ 6） | 遗漏 ≫ 7 → 超期冷号（回补候选） |
| 后区单号出现次数 | ≈ 8 次（50×2÷12） | 判断后区冷热 |
| 后区单号平均出现间隔 | ≈ 6 期（平均遗漏 ≈ 5） | 遗漏 ≫ 6 → 后区回补候选 |
| 单期前区三区间球数 | 各 ≈ 1.7 / 1.7 / 1.6 个 | 区间均衡判断 |
| 前区和值 | 期望 ≈ 90，常见 60–120 | 和值回归判断 |

> 这些是「锚点」。视觉观察发现的偏离，都要拿这些期望值来度量「偏离了多少」。

## 1. 读图前的定向（先确认坐标系）

读取 PNG 后，先在心里建立坐标：

- **行 = 期号**，**最上面是最旧期，最下面是最新期**（基准期在最底行）。
- **列 = 号码**：前区 01–35 分三块（01–12 / 13–24 / 25–35），右侧蓝色区为后区 01–12。
- **实心红球/蓝球 = 当期中出**；**灰色小数字 = 遗漏值**（距上次出现已隔多少期）。
- 预测目标 = **最底行再往下一行**（下一期）。趋势线的「延长落点」就指向这条虚拟新行。

---

## 2. 五步观察协议（按顺序执行，每步都要"说出所见"）

为避免凭空臆断，每一步都要**先用文字描述在图上实际看到了什么**，再下结论。

### Step A · 总览带状结构（宏观）

整体扫一遍红球分布，找出：
- **密集带**：红球扎堆的列区或斜向条带（近期热区）。
- **空白带**：大片只有灰色遗漏数字、几乎没有红球的列区（近期冷区）。
- 后区蓝球同样看密集列与空白列。

输出：用一句话描述「热区在哪、冷区在哪」。

### Step B · 纵向单号节奏（冷热与回补）

逐区扫视每个号码列底部的灰色遗漏数字（越靠下越新）：

- **超期冷号**：当前遗漏值明显大于平均间隔（前区 > 12、后区 > 10 即显著），属于「欠债」号码，回归压力大 → **回补候选**。
- **热延续号**：最近 3–5 期内反复出现、间隔很短的号码 → 可能延续，但也可能透支回调（结合 Step D 判断）。
- **正常波动号**：遗漏在 0–7 之间规律跳动，权重低。

输出：列出 3–6 个超期冷号 + 2–4 个近期热号（含其当前遗漏值）。

### Step C · 斜向趋势线（连线法，本程序的核心）

这是参考图中**绿色箭头**所做的事：把相邻若干期的红球用斜线连接，识别号码沿对角方向的运动。

观察四类线：

1. **右下斜带（↘）**：连续几期的球，号码逐期增大（位置向右下移动）。延长线指向下一期更大的号码。
2. **左下斜带（↙）**：连续几期的球，号码逐期减小（向左下移动）。延长线指向下一期更小的号码。
3. **交叉/收敛点（X 形）**：一条上行线与一条下行线相交，交点附近的号码区是**近期反复命中的"枢纽区"**，下一期常在交点延伸的列附近出号。
4. **平行通道**：多条同向斜线形成的带状通道，号码沿通道整体平移，下一期大概率仍落在通道延长区间。

操作要点：
- 优先连**最近 8–12 期**的球，趋势线越靠近底部越有预测价值；远端老线仅作背景。
- 一条有效趋势线至少要串起 **3 个及以上**期的球，避免两点连线的偶然。
- 把每条趋势线的「延长落点列号」记下来——这是前区候选的重要来源。
- 后区列窄，主要看蓝球是否形成明显上行/下行的短斜线。

输出：描述 2–4 条主要趋势线（方向 + 串联的大致期 + 延长落点号码区间）。

### Step D · 区间均衡（回归压力）

统计**最近 3–5 期**前区三区间各出了几个球，与期望（每区每期 ≈ 1.7 个）对比：

- **欠债区间**：连续几期出球明显偏少（如某区连续 3 期只出 0–1 个）→ 下一期回补概率上升。
- **超额区间**：连续几期出球偏多（如某区连续 3 期出 3 个）→ 下一期回调概率上升。
- 同理看**奇偶比、大小比（小区 01–17 / 大区 18–35）、和值走向**：若最近几期持续偏向一端（如连续偏小、和值持续偏低），按回归预期下一期向中间值靠拢。

输出：指出 1–2 个欠债区间 / 超额区间 + 和值近期走向（偏高/偏低/居中）。

### Step E · 形态辅助（次要权重）

- **连号**：近期是否频繁出连号（如 12-13）。
- **重号**：下一期与上一期重复号的概率（前区平均每期 ≈ 0.7 个重号，即约 7 成期有 1 个重号）。
- **尾数分布**：和尾列是否在某些尾数上偏离。

输出：一句话补充形态线索。

## 3. 综合定号（从观察到预测）

把 Step B–D 的候选汇总、加权、收敛：

### 前区（固定选 7 个）

1. **建立候选池（12–16 个）**：超期冷号（回补）+ 趋势线延长落点号 + 欠债区间内的号 + 必要的热延续号。
2. **加权收敛**：
   - 同时满足「超期冷号 + 落在欠债区间 + 有趋势线指向」的号 → 最高优先级。
   - 仅满足单一条件的号 → 次优。
3. **结构校验**：最终 **7 个号**尽量满足——三区间覆盖 ≥2 区（避免 7 球全挤一区）、大小比不极端、和值合理、奇偶不极端（避免全奇/全偶）。
4. **固定输出 7 个前区号**（不再区分核心/备选，统一为 7 码定号池）。

### 后区（固定选 3 个）

- 后区候选 = 后区超期冷号 + 蓝球短斜线落点。
- **固定输出 3 个后区号**。

## 5. 常见误区（务必规避）

- ❌ **不看图就报号**：必须基于 PNG 视觉观察，先描述所见再定号。
- ❌ **两点连线当趋势**：趋势线至少串 3 期，否则是噪声。
- ❌ **只追热号**：均态分布的核心是回归，冷号回补与热号延续要平衡，不能 7 码全是热号或全是冷号。
- ❌ **过度自信**：不得宣称"必中""高命中率"。如实声明随机性与娱乐属性。
- ❌ **结构失衡**：避免 7 球全在一个区间、全大/全小、全奇/全偶、和值极端。
- ❌ **数量不符**：最终定号必须恰好 **前区 7 个 + 后区 3 个**，不多不少。
