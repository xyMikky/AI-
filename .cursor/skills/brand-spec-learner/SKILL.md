# 品牌规范学习引擎（Brand Spec Learner）

## 概述

品牌素材的全自动学习与规范沉淀引擎。前身是能力模块 M17_品牌规范学习引擎。

**与 reference-library-learner 的边界**：本 Skill 处理品牌专属素材（写入 `品牌规范/[品牌名]/`）；后者处理参考库通用素材（写入 `参考库/`）。

## 触发条件

| 用户语义 | 是否激活 |
|---|---|
| "学习品牌 [品牌名]" | ✅ |
| "初始化品牌 [品牌名]" / "创建品牌 [品牌名]" | ✅ 新建模式 |
| "更新品牌规范" / "学习品牌素材" | ✅ |
| "学习这张图"（参考素材） | ❌ → reference-library-learner |

**M13 豁免**：学习任务流程固定，跳过需求引导。

## 核心流程

```
触发学习品牌 [品牌名]
  ↓
Step 1：检查品牌目录
  ├─ 已存在 → 进入增量学习模式
  └─ 不存在 → 从 `品牌规范/_品牌目录模板/` 复制初始化
  ↓
Step 2：扫描 `品牌规范/[品牌名]/00_待学习/`
  ↓
Step 3：按格式分类
  ├─ 图片 → 视觉资产（设计案例）
  ├─ PDF / Word → 提取品牌规范文本
  ├─ 文本 → 品牌语气 / 禁忌清单
  └─ 字体文件 → 字体系统
  ↓
Step 4：写入对应视觉系统文件（按视觉维度）
  ├─ 色值/色板 → `视觉系统/色彩系统.txt`
  ├─ 字体 → `视觉系统/字体系统.txt`
  ├─ Logo 用法 → `视觉系统/Logo系统.txt`
  ├─ 影像风格 → `视觉系统/影像风格.txt`
  ├─ 版式规范 → `视觉系统/版式规范.txt`
  ├─ 品牌语气 → `视觉系统/品牌语气.txt`
  └─ 禁忌清单 → `视觉系统/禁忌清单.txt`
  ↓
Step 5：归档设计案例
  → `品牌规范/[品牌名]/设计案例/[类别]_参考记录.txt` 追加记录
  → 案例图复制到 `设计案例/`，命名 `[品牌名]-[类型]-[序号]-[关键词].ext`
  ↓
Step 6：更新品牌档案
  → `品牌规范/[品牌名]/品牌档案.txt`（核心档案，<3KB）
  → `品牌规范/[品牌名]/品牌索引.txt`（内导航）
  → `品牌规范/品牌规范主控.txt`（品牌注册表）
  ↓
Step 7：清理已处理素材
  → 移除 `00_待学习/` 中已处理的文件，保留 .txt 说明
```

## 品牌目录结构（模板）

```
品牌规范/[品牌名]/
├── 品牌档案.txt              ← always-load, <3KB
├── 品牌索引.txt              ← 内导航
├── 00_待学习/                ← 新素材入口
├── 设计案例/                  ← 已学习的案例
│   ├── 电商_参考记录.txt
│   ├── 广告投放_参考记录.txt
│   └── 详情页_参考记录.txt
└── 视觉系统/                  ← 7 个结构化规范文件
    ├── 色彩系统.txt
    ├── 字体系统.txt
    ├── Logo系统.txt
    ├── 影像风格.txt
    ├── 版式规范.txt
    ├── 品牌语气.txt
    └── 禁忌清单.txt
```

## 与其他 Skill 的关系

| Skill | 关系 |
|---|---|
| `reference-library-learner` | 互斥：参考库 vs 品牌库 |
| `brand-logo-designer` | 互补：本 Skill 学已有 Logo；前者从零创建 |
| `ai-image-prompt-builder` | 下游消费：品牌激活时读取本 Skill 沉淀的色彩/字体/版式 |
| `non-ai-design-reviewer` | 协同：评审品牌物料时读品牌档案 |
| `brand-spec-loading.mdc` | 配套加载规则（V8.0 阶段 4 时合并入本 Skill） |

## 加载约定

品牌被激活后，**默认加载**：
- `品牌档案.txt`（强制，<3KB 极轻量）

**按任务类型加载**：
| 任务 | 加载文件 |
|---|---|
| 海报/Banner/电商图 | + 色彩系统 + 字体系统 + 影像风格 |
| Logo/品牌设计 | + Logo系统 + 色彩系统 |
| 详情页 | + 版式规范 + 字体系统 + 色彩系统 |
| 视频/动效 | + 影像风格 + 品牌语气 + 色彩系统 |
| 所有任务 | + 禁忌清单（与 Z 禁忌档案合并加载） |

## 注意事项

- 品牌档案.txt 必须 <3KB，超出说明信息密度过低，应拆到对应视觉系统文件
- 品牌名是模型未知 token，**Prompt 中不得放在内容描述句**（详见 prompt-hygiene 铁律）
- 品牌规范激活时优先级 > 参考库个人偏好（三轴优先级）
- 完整流程详见 `归档/v7.9/能力模块/M17_品牌规范学习引擎.txt`
