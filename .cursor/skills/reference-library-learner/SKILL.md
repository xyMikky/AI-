# 参考库学习引擎（Reference Library Learner）

## 概述

参考库素材的全自动分类学习与归档引擎。前身是能力模块 M10_自主学习引擎（V4.0 通识层双轨写入版）。

**与 brand-spec-learner 的边界**：本 Skill 处理 `参考库/00_待学习/`（个人审美素材）；后者处理 `品牌规范/[品牌名]/00_待学习/`（品牌专属资产）。

## 触发条件

| 用户语义 | 是否激活 |
|---|---|
| "开始学习" / "继续学习" | ✅ |
| "处理待学习" / "扫描待学习文件夹" | ✅ |
| "更新参考库" | ✅ |
| "学习这张图"（用户上传单张） | ✅ 单图模式 |
| "学习品牌素材" | ❌ → brand-spec-learner |

**M13 豁免**：学习任务流程固定，跳过需求引导直接执行。

## 核心流程

```
触发学习
  ↓
Phase 0.5 预细分清理 ← 扫描全域子文件，对 ≥12 条记录的子文件先细分
  ↓
Phase 1 列文件 ← 用 .NET IO 列 00_待学习/（中文路径必须）
  ↓
Phase 2 复制到临时目录 + 重命名 img_1/img_2... + Read 分析
  ↓
Phase 3 写入对应域子文件 ← 写入后立即检查记录数，≥12 则同步细分
  ↓
Phase 3.5 通识层双轨写入 ← 与 P1-P7 通识原则交叉匹配
  ↓
Phase 4 归档 ← 复制到 [域]/原始素材/[规范命名].ext，清空 00_待学习/
  ↓
Phase 5 同步三层索引 ← 子文件头 / X_域索引.txt / 参考索引.txt
  ↓
Phase 5.5 兜底验证 ← 确保所有子文件 < 12 条
  ↓
Phase 6 向量索引增量更新 ← python 工具/向量检索/update_index.py
```

## 域分类规则

| 域 | 路径 | 内容 |
|---|---|---|
| A | `参考库/A_品牌与Logo/` | Logo、VI |
| B | `参考库/B_配色系统/` | 色值/色板 |
| C | `参考库/C_排版字体/` | 字体/排版 |
| D | `参考库/D_UI界面/` | UI 截图 |
| E | `参考库/E_平面与海报/` | Banner/海报/电商主图 |
| F | `参考库/F_摄影风格/` | 摄影 |
| G | `参考库/G_插画与图形/` | 插画/图标 |
| H | `参考库/H_3D与渲染/` | 3D/CGI |
| I | `参考库/I_动效与视频/` | 视频/动效 |
| J | `参考库/J_空间与产品/` | 包装/空间 |
| L | `参考库/L_口播文案库/` | 口播脚本 |
| N | `参考库/N_详情页设计/` | 详情页（独立于 E） |
| V | `参考库/V_视频脚本库/` | 短视频结构/分镜 |
| Z | `参考库/Z_禁忌案例/` | 反面案例 |

**详情页特征**（路由到 N 域而非 E 域）：长图、含产品参数、模块化版式、文件名含"详情/listing/A+/DP"。

## 学习记录硬性格式

```
--- 记录 #[序号] · [YYYY-MM-DD] ---
来源：M10自主学习 · [归档文件名]（[品牌/产品信息]）
关键标签：[标签1] / [标签2] / ...
喜爱程度：★★★★☆
场景标签：[K1_xxx] / [K2_xxx] / [K3_xxx] / [K4_xxx]

【[亮点标题：根据素材实际特点自由命名]】
[分析内容]

【可迁移原则】  ← 必填，所有记录均须包含
[在什么设计场景下可应用，具体怎么用]
---
```

- 【可迁移原则】硬性必须
- 字段名不加 Markdown 加粗
- 分隔线用 `---` 不用 `###`

## 归档命名规则

`[域代码]-[序号]-[关键词]-[地区]-[子域]-[日期].[扩展名]`

例：`E-031-Popilush-RedLaceBodysuit-DiagonalSplit-US-EC-20260323.png`

## 路径技术约束（中文路径）

```powershell
# ✅ 必须用 .NET IO（PowerShell Get-ChildItem 中文路径不可靠）
$root = $PWD.Path
$src = Join-Path $root "参考库\00_待学习"
[System.IO.Directory]::GetFiles($src) | Where-Object { $_ -notlike "*.txt" }

# ❌ 禁用
# Get-ChildItem "参考库\00_待学习"
```

## 与其他 Skill 的关系

| Skill | 关系 |
|---|---|
| `brand-spec-learner` | 互斥：处理参考库 vs 品牌库 |

## 与向量索引的协同

每完成一批学习后必须执行：
```powershell
python 工具/向量检索/update_index.py
```

确保新增切片可被 `search.py` 召回。

## 注意事项

- 路径含中文必须用 .NET IO 方法
- 「待分类」目录中未命名素材不入索引，调用时忽略
- 学习不激活任何设计模块（M1-M9 等），专注档案维护
- 完整流程详见 `归档/v7.9/能力模块/M10_自主学习引擎.txt`
